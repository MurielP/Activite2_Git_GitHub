# Exercice pratique  - Git & GitHub

1 - Présenter le projet dand un fichier README.txt
2 - Ajouter 2 fichiers 
3 - Réaliser 4 commits avec des messages qui expliquent précisément les modifications apportées 
4 - Envoyer le repo Git sur GitHub
5 - Copier l'URL https de ce repo GitHub créé pour cet exercice
6 - Télécharger le document texte et indiquer son URL dans ce document. Enregistrer
7 - Compresser le document .txt enregistré dans un fichier .zip et envoyer ce fichier sur OC
